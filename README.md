# DS4420-airbnb-final-project
## Project Overview
This project compares neural networks and Bayesian regression models in predicting Airbnb listing prices and capturing uncertainty in price estimates.

## Research Question
How do neural networks and Bayesian regression models compare in predicting Airbnb listing prices and capturing uncertainty in price estimates?

## Dataset
Inside Airbnb, New York City listings data (November 07, 2025 snapshot).
Download source: https://insideairbnb.com/get-the-data/

## Models
1. OLS baseline
2. Neural network
3. Bayesian regression

## Phase I Deliverable
Phase I includes:
- literature review
- GitHub repository setup
- proof-of-concept for the first non-Bayesian model

## Repository Structure
- `data/`: dataset notes or data access instructions
- `notebooks/`: exploratory work and proof-of-concept models
