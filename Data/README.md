# data

Primary dataset: Inside Airbnb, New York City listings data  
Snapshot date: November 07, 2025

Download source: https://insideairbnb.com/get-the-data/

Place the downloaded listings file in this folder before running the notebook.

This project uses the Inside Airbnb New York City listings dataset, which contains detailed information about Airbnb listings in New York City. The dataset includes listing characteristics, host information, review metrics, and location attributes that can be used to analyze and predict Airbnb listing prices.

Inside Airbnb provides publicly available datasets that are compiled from Airbnb listing pages. These datasets are widely used for research on short-term rentals, housing markets, and pricing models.

The dataset contains information about individual Airbnb listings in New York City. Each row represents one listing, and the columns describe listing characteristics, host attributes, and review metrics.

# Examples of variables used in this project include:

Listing characteristics：

price – nightly listing price

accommodates – number of guests the listing can host

bedrooms – number of bedrooms

bathrooms_text – bathroom description

property_type – type of property (e.g., apartment, house)

Location information：

neighbourhood_cleansed – NYC neighborhood

latitude, longitude – geographic coordinates

Host attributes：

host_is_superhost – whether the host is a Superhost

host_listings_count – number of listings owned by the host

Review information：

number_of_reviews – total review count

review_scores_rating – overall rating score
